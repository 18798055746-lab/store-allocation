## 快速启动

### 1. 配置飞书

参考 `FEISHU_SETUP.md` 配置飞书应用，获取 App ID 和 App Secret。

### 2. 填写环境变量

编辑 `server/.env` 文件：

```env
FEISHU_APP_ID=你的App ID
FEISHU_APP_SECRET=你的App Secret
```

### 3. 启动系统

**方式一：使用启动脚本（推荐）**

双击 `start.bat`，自动安装依赖并启动服务。

**方式二：手动启动**

```bash
# 终端1：启动后端
cd server
npm install
npm run dev

# 终端2：启动前端
npm install
npm run dev
```

### 4. 访问系统

- **学员端**：http://localhost:5173
- **管理员端**：http://localhost:5173/admin

1. **类型匹配**
   - 服务类学员只能选服务类门店

## 常见问题

### Q: 如何修改门店容量？

A: 直接在飞书多维表的"门店信息表"中修改"门店可容纳人数"字段。

### Q: 如何查看分配结果？

A: 访问管理员端 `/admin`，切换到"分配详情"标签页。

### Q: 门店满了怎么办？

A: 系统会自动标记为"已满"，学员无法选择已满的门店。

### Q: 如何重置分配？

A: 在飞书多维表的"集训信息收集表"中删除相关记录。
