# 门店实训志愿分配系统

## 功能介绍

这是一个用于2026年中国区校招生培训的门店实训志愿分配系统。

### 业务背景

- **90位学员**：分为服务类（20人）和非服务类（74人）
- **51家门店**：
  - 服务类：9家门店（2家各容纳3人，7家各容纳2人）
  - 非服务类：42家门店（每家容纳2人）
- **志愿机制**：每人可选择3个志愿，按顺序分配
- **实时更新**：门店剩余名额实时显示

### 技术架构

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   React前端     │ ──▶ │  Node.js后端    │ ──▶ │  本地JSON文件   │
│  (Vite+Tailwind)│    │  (Express)      │    │  (数据存储)     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

## 快速开始

### 1. 安装依赖

```bash
# 前端依赖
npm install

# 后端依赖
cd server
npm install
```

### 2. 配置学员数据

编辑 `server/data/students.json` 文件，添加学员信息：

```json
{
  "S001": {"id": "S001", "name": "张三", "type": "服务"},
  "S002": {"id": "S002", "name": "李四", "type": "非服务"}
}
```

### 3. 启动服务

```bash
# 方式一：使用启动脚本（推荐）
start.bat

# 方式二：手动启动
# 终端1：启动后端
cd server
npm run dev

# 终端2：启动前端
npm run dev
```

### 4. 访问系统

- 学员端：http://localhost:5173
- 管理员端：http://localhost:5173/admin

## 功能说明

### 学员端功能

1. **身份验证**：输入工号和姓名进行验证
2. **门店选择**：查看所有可选门店及实时余量
3. **志愿排序**：选择最多3个志愿，按优先级排序
4. **实时反馈**：门店状态实时更新（每10秒刷新）
5. **分配结果**：提交后立即显示分配结果

### 管理员端功能

1. **数据概览**：总门店数、总容量、已报名、剩余名额
2. **分类查看**：服务类/非服务类门店分别展示
3. **分配详情**：查看每个门店的学员分配情况
4. **实时刷新**：手动刷新最新数据
5. **导出Excel**：一键导出分配结果Excel文件

## 数据文件说明

### 门店数据 `server/data/stores.json`

```json
{
  "service": [
    {"name": "门店名称", "capacity": 3, "type": "服务"}
  ],
  "nonService": [
    {"name": "门店名称", "capacity": 2, "type": "非服务"}
  ]
}
```

### 学员数据 `server/data/students.json`

```json
{
  "S001": {"id": "S001", "name": "张三", "type": "服务"},
  "N001": {"id": "N001", "name": "李四", "type": "非服务"}
}
```

### 分配结果 `server/data/allocations.json`

系统自动维护，包含所有分配记录。

## Excel导出

管理员端点击"导出Excel"按钮，会生成包含以下3个工作表的Excel文件：

1. **分配结果**：工号、姓名、类型、分配门店、志愿顺序、分配时间
2. **门店汇总**：门店名称、类型、容量、已分配、剩余、状态
3. **学员名单**：工号、姓名、类型、分配门店、志愿顺序

## 并发控制

系统实现了多重并发控制机制：

1. **前端轮询**：每10秒更新门店余量
2. **后端锁**：防止同一学员重复提交
3. **原子操作**：分配操作具有原子性
4. **文件持久化**：数据实时保存到JSON文件

## 常见问题

### Q: 如何修改门店容量？

A: 编辑 `server/data/stores.json` 文件，修改对应门店的 `capacity` 字段。

### Q: 如何添加学员？

A: 编辑 `server/data/students.json` 文件，按照格式添加学员信息。

### Q: 如何查看分配结果？

A: 访问管理员端 `/admin`，切换到"分配详情"标签页。

### Q: 如何导出Excel？

A: 在管理员端点击"导出Excel"按钮即可下载。

### Q: 门店满了怎么办？

A: 系统会自动标记为"已满"，学员无法选择已满的门店。

### Q: 如何重置分配？

A: 删除 `server/data/allocations.json` 文件中的内容，或清空该文件。

## 技术栈

- **前端**：React 18 + Vite + TailwindCSS
- **后端**：Node.js + Express
- **存储**：本地JSON文件
- **导出**：SheetJS (xlsx)
